# Google Workspace User Management

## Overview
Managed user accounts and access in Google Workspace to ensure secure, role-based access to systems and sensitive data within a human services environment.

## Responsibilities
- Created and managed user accounts
- Assigned roles and permissions based on job function
- Configured email and system access
- Resolved authentication and login issues
- Updated and revoked access as needed

## Security & Compliance
- Enforced role-based access control (RBAC)
- Applied least privilege to limit unnecessary access
- Verified user identity before granting or changing access
- Maintained confidentiality of account information
- Followed consistent account management procedures

## Controls & Practices
- Access provisioning and deprovisioning
- Permission reviews and updates
- Account lifecycle management
- Documentation of changes and actions

## Tools
- Google Workspace Admin Console

## Skills
- Identity and Access Management (IAM)
- Access Control
- User Administration
- IT Support
